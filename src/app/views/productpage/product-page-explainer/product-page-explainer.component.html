Kubera Cloud Edition

Kubera Propel and Chaos are lightweight and easy to run yourself. But, what if you want MayaData to run Kubera for you? Then select Kubera Cloud Edition. With Kubera Cloud Edition you can see and configure all of your OpenEBS deployments, set up back-up services to any S3 compliant object store, and simply map and manage your workloads to your underlying storage services. Kubera Cloud Edition even includes off cluster logging, charting, and alerts for OpenEBS and for any of your stateful workloads.

Please note that currently Litmus Chaos support is not available in Kubera Cloud Edition.
